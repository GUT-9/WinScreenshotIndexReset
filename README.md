# WinScreenshotIndexReset

**中文**  
一键重置 Windows 自带截图（Win+PrtSc）文件计数器，省去手动翻注册表的麻烦。  
若注册表值不存在，脚本不会做任何修改，可放心使用。

**English**  
One-click reset the built-in screenshot (Win+PrtSc) file index counter on Windows—no more digging through the registry.  
If the registry value is not found, the script will leave everything untouched; use it with confidence.
