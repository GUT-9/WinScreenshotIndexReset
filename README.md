# WinScreenshotIndexReset
将Win自带的屏幕截图（Win+PrtSc）的截图文件计数一键重置，省去了自己去注册表找值的步骤，没有找到注册表值则不会进行任何更改，可放心使用。
One-click reset the screenshot file count of the built-in screenshot function (Win + PrtSc) in Windows, eliminating the need to search for values in the registry by yourself. If the registry value is not found, no changes will be made, so you can use it with confidence.
